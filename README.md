Vi Sync is a simple program made to copy files between two folders and sync them together

Installation: 

  1. Ensure that you have .NET 8.0 installed (you can check if you do by running `dotnet --list-runtimes` in your terminal)  
  If you don't have it, download from [here](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)

  2. Download ViSync.zip or ViSync-Linux.zip (coming soon) from the latest [release](https://github.com/silly-Violet/ViSync/releases/), extract the contents and run ViSync.exe / ViSync.sh (coming soon)
